description Features

Dility is a small web application, whose main feature is calculating daily efficiency of employees. Application calculates efficiency based on tasks, whose employees performed during workday. Every task has given time to perform, actually application supports three example tasks (add, delete, move). Data is stored in MSSQL database and as csv, json files stored in "Data" folder. Initially there is created manually data from 14-03-2020.

Workdays are created manually, by filling form in "Add Workday" tab - choosing date, employees working that day and tasks their performed.Workday is saved in database and proper files are generated. Insertion of new employees is possible in "Add Employees" tab.

In "Workdays" are displayed existing workdays. Application automatically finds last workday and calculates efficiency and few other things about actions performed by employees. We can change day with data picker.
